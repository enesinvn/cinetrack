# Ekran Görüntüleri

Uygulamayı `npm run dev` ile çalıştırdıktan sonra:

1. Tarayıcıda http://localhost:5173 adresine gidin
2. Birkaç film/dizi ekleyin ki ekran dolu görünsün
3. Ekran görüntüsünü alıp bu klasöre `home.png` adıyla kaydedin
4. README.md'deki referans (`./screenshots/home.png`) otomatik çalışacaktır

## Önerilen Ekran Görüntüleri

- `home.png` — Ana sayfa (tüm filmler listesi)
- `add.png` — Yeni film ekleme formu (opsiyonel)
- `watched.png` — İzlenenler sayfası (opsiyonel)
